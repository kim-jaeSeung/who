프로젝트 설명

여행 계획 간소화 복잡한 일정 계획 없이 추천과 검색으로 여행 일정을 구성할 수 있도록 설계를 했습니다. 국내 여행 독려 공공데이터 기반의 추천과 지도 검색을 통해 다양한 국내 명소를 소개

📌 상세 내용

- 로그인 / 회원가입 (JWT 인증)

사용자 인증 기능 구현.

JWT를 활용하여 보안성과 확장성 있는 인증 체계 구축.

- 지역명 검색 기능

사용자가 원하는 지역명을 직접 검색하여 여행지를 탐색할 수 있도록 구현

- 커뮤니티 기능

여행 팁, 일정 공유 등을 주제로 자유롭게 글을 작성하고 읽을 수 있는 게시판

CRUD(생성, 조회, 수정, 삭제) 기능 완비

- 랜덤 여행지 추천 (룰렛 기능)

룰렛 형태의 인터랙션을 통해 랜덤으로 국내 여행지를 추천

사용자 경험(UX)을 고려한 재미 요소 추가

- 확정된 여행 계획 저장

사용자가 여행 일정을 선택 및 확정하고 저장할 수 있는 기능

이후 커뮤니티 공유 및 개인 보관 가능

- KakaoMap 기반 장소 검색

카카오맵 API를 활용하여 사용자 입력에 따라 여행지, 맛집 등을 검색

마커 및 지도 뷰 연동

👍 프로젝트 성과 및 배운 점

- React 기반 SPA 구조 및 컴포넌트 설계 학습 - GitHub, Figma 등 협업 도구를 통한 체계적인 개발 진행 - 팀장으로서 프로젝트 관리 및 팀원 간의 원활한 소통 경험

✨ 랜덤여행지추천

프로젝트 설명

이 프로젝트는 사용자가 특정 지역(시도, 시군구)을 선택하거나 지역명을 검색하여 여행지를 추천받을 수 있는 웹 기반 여행지 추천 앱입니다. 단순한 추천을 넘어, 여행 계획 수립과 사용자 커뮤니티 기능으로 확장 가능한 여행 도우미 플랫폼을 목표로 개발 중입니다.

관련 링크

Git hub

📌 상세 내용

- 시도/시군구 기반 여행지 추천 시도와 시군구를 선택하면, 해당 지역의 여행지 중 하나를 랜덤으로 추천합니다. 행정구역 정보 및 여행지 데이터는 공공 API를 기반으로 합니다. - 지역명 검색 기능 (개발 중) 사용자가 원하는 지역명을 직접 검색하여 여행지를 탐색할 수 있도록 구현 예정입니다. - 콘텐츠 기반 추천 (기획 단계) 예: "자연 + 카페", "역사 + 맛집" 등 두 가지 키워드 조합으로 여행지 추천

⚠️ 현재 한계점

- 여행지 상세 내용 부족

공공 API에서 상세 설명이 제공되지 않아 보류 상태 / 일부 데이터르 만들어서 사용 중

- 이미지 데이터 누락

API 응답에 이미지가 없는 경우가 다수 존재함

현재는 이미지가 있는 여행지만 우선 활용

✨ 풉타임

프로젝트 설명

사용자가 화장실에서 시간을 보내는 동안 지루함을 달래고 즐거움을 얻을 수 있는 유익한 모바일 애플리케이션을 개발합니다. 스마트폰을 사용하는 모든 연령대의 사용자, 특히 화장실에서 스마트폰을 자주 사용하는 사용자를 타겟으로 합니다.

📌 상세 내용

- 해당 프로젝트 같은 경우에는 팀원들이 각자 본인들의 방법으로 모든 기능을 구현하여 진행했습니다.

기능별로 각각의 팀원들이 구현한 기능을 투표나 팀장의 결정으로 프로젝트에서 사용하는 방향으로 진행했습니다

- 재미를 줄 수 있는 콘텐츠

퀴즈, 상식, 괴담, 유머 등 다양한 콘텐츠를 제공하여 사용자가 화장실에서 시간을 보내는 동안 즐거움을 느낄 수 있도록 합니다.

- 커뮤니티 기능

사용자들 간의 소통을 통해 좀 더 재미를 유발할 수 있는 커뮤니티 기능을 제공합니다.

- 로그인 기능

로그인 기능을 통해 사용자는 개인화된 콘텐츠를 즐길 수 있습니다.

👍 프로젝트 성과 및 배운 점

- 직접 가공한 데이터를 사용하여 API를 통해 콘텐츠를 제공하는 방법을 배웠습니다.

퀴즈,상식,괴담 데이터를 직접 가공하여 Rest API를 통해 콘텐츠를 제공

- 팀원들과의 협업을 통해 다양한 기술 스택을 활용한 개발 경험을 쌓았습니다.

새로운 기술 스택을 도입하며 팀원들과 적극적으로 소통하고 함께 학습하면서 프로젝트를 완수했습니다.
