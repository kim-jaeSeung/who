# Portfolio Renewal

This project has been renewed using **React**, **TypeScript**, and **Tailwind CSS**.

## Project Structure

- **`src/data/portfolio.ts`**: **Edit this file** to add/remove portfolio items.
- **`src/components/`**: UI Components.
- **`public/img/`**: Images.

## How to Run

1.  `npm install` (already done)
2.  `npm run dev` (to start local server)
3.  `npm run build` (to build for deployment)

## How to Add Portfolio Items

1.  Add your image to `public/img/`.
2.  Open `src/data/portfolio.ts`.
3.  Add a new item to the `portfolioData` array.

## Legacy

Old files are in `legacy_backup/`.
